# 【觅长生】MCS Multi Cheats

## 一、介绍

本模组是一个由二十多个功能组成修改合集，涵盖个人喜好的全部修改内容，欢迎大家试用。

代码基于最新版本开发，遵循MIT协议，少部分内容参考和引用现有模组，详见鸣谢，如有侵权问题请及时联系我调整。

## 二、功能说明

1. 神秘绿瓶: 解锁全部药性，天赋点`+12`，随机生成灵气`+N^2`，灵田灵石效果`xN^2`，每月催生速度`xN^2`
2. 道心坚定: 论道灵感消耗`/8`，最大回合`+N^2`、每轮空节点`+1`，悟道速率、悟道经验、悟道值受【悟性+资质+灵感】加成`x1~10`
3. 耐药调整: 道源丹、道源丹耐药`8`，耐药Perk`x4`，突破四大境界重置耐药性
4. 炼丹调整: 丹炉草药上限`xN`，丹圣Perk`xN`，全部药材可种植（妖丹除外）
5. 炼器调整: 时间消耗`N`，材料灵气`xN`，材料权重`x4`
6. 其它调整: 拍卖会数量修改`3|12`，物品最低六成掉落，论道最大悟道点`480`，命犯桃花异性送礼好感加成`xN`
7. 交互优化: 战斗右键快捷施法，论道池滚动显示，传音符物品自动领取，人物修为显示（侧边栏、传音符）

**已知问题**

- [ ] 游玩过程中可能存在卡死情况，重启后恢复，由于我加了其它第三方模组，不确定是否由本模组导致的。
